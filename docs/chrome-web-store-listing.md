### 詳細説明

```
競馬データを一瞬で AI 分析用に変換する Chrome 拡張機能です。

【こんな方におすすめ】
- ChatGPT や Claude で競馬予想をしている方
- netkeiba のデータを手動でコピーするのが面倒な方
- Excel や Google Sheets で競馬データを分析したい方
- Python やスクレイピングの知識がない方

【主な機能】
- ワンクリックでレースデータを抽出
- 3 つの出力フォーマット: Markdown / CSV / AI 分析プロンプト
- netkeiba.com（出馬表・レース結果）に対応
- JRA 公式サイト（出馬表・レース結果）に対応

【使い方】
1. netkeiba.com または JRA 公式の出馬表/レース結果ページを開く
2. 拡張機能のアイコンをクリック
3. フォーマットを選択（Markdown / CSV / AI）
4. 「データ抽出」をクリック
5. 「クリップボードにコピー」で完了
6. ChatGPT、Claude、Excel などに貼り付けて分析

【Free プラン】
- Markdown テーブル出力（無料・無制限）

【Pro プラン】
- CSV 出力（Excel ・ Google Sheets 対応）
- AI 分析プロンプト自動生成（予想用・回顧用）

【プライバシー・安全性】
- DOM 読み取りのみ。データは外部送信されません
- サーバーとの通信は一切ありません
- ライセンス情報もローカル保存のみ

【対応サイト】
- netkeiba.com（出馬表・レース結果）
- JRA 公式サイト（出馬表・レース結果）

ソースコード: https://github.com/bbtc3453/keiba-race-data-formatter
```

### Detailed Description

```
A Chrome extension that instantly converts Japanese horse racing data into AI-ready formats.

【Who is this for?】
- Anyone using ChatGPT or Claude for horse racing predictions
- International fans analyzing JRA races (Japan Cup, Arima Kinen, etc.)
- Data analysts building racing prediction models
- Anyone tired of manually copying race data from netkeiba.com

【Features】
- One-click race data extraction
- 3 output formats: Markdown / CSV / AI analysis prompt
- Supports netkeiba.com (race cards & results)
- Supports JRA official site (race cards & results)

【How to use】
1. Open a race card or result page on netkeiba.com or JRA official site
2. Click the extension icon
3. Select your format (Markdown / CSV / AI)
4. Click "Data Extract"
5. Click "Copy to Clipboard"
6. Paste into ChatGPT, Claude, Excel, or any tool

【Free Plan】
- Markdown table output (free, unlimited)

【Pro Plan】
- CSV export (Excel & Google Sheets compatible)
- AI analysis prompt auto-generation (prediction & review templates)

【Privacy & Safety】
- DOM read-only. No data is sent externally
- Zero server communication
- License info stored locally only

【Supported Sites】
- netkeiba.com (race cards & results)
- JRA official site (race cards & results)

Source code: https://github.com/bbtc3453/keiba-race-data-formatter
```
